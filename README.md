# ROT13 Šifrēšanas un Atšifrēšanas Programma

Šī programma ir izveidota, lai veiktu ROT13 šifrēšanu un atšifrēšanu, kas ir vienkāršs alfabēta burtu pārbīdes algoritms. Programma atbalsta latviešu un angļu valodas.

---

## Funkcionalitāte
- **Divvalodu atbalsts**: Lietotājs var izvēlēties starp latviešu un angļu valodām.
- **Šifrēšana un atšifrēšana**: ROT13 algoritms ļauj apstrādāt tekstu, lai to šifrētu vai atšifrētu.

---

## Prasības
Lai palaistu šo programmu, nepieciešams:
- **Python 3.x** instalācija jūsu datorā

---

## Uzstādīšana un Palaišana

1. **Klonējiet repozitoriju vai saglabājiet skriptu**:
   Saglabājiet zemāk redzamo kodu failā ar nosaukumu `rot13.py`.

2. **Palaidiet programmu**:
   ```bash
   python rot13.py
   ```

---

## Lietošanas instrukcija

1. **Izvēlieties valodu**:
   - Ievadiet `1`, lai izvēlētos latviešu valodu.
   - Ievadiet `2`, lai izvēlētos angļu valodu.

2. **Izvēlieties darbību**:
   - Ievadiet `1`, lai šifrētu tekstu.
   - Ievadiet `2`, lai atšifrētu tekstu.

3. **Ievadiet tekstu**:
   Kad programma prasa, ievadiet tekstu, kuru vēlaties apstrādāt.

4. **Rezultāts**:
   Programma parādīs apstrādātā teksta rezultātu ekrānā.
